Bienvenidos a WEBMEDICALVET!

esta aplicacion web trata de que el profesional medico veterinario pueda cargar los datos de sus pacientes a una tabla predefinida para luego guardarlo en la aplicacion o pueda bajarlo a su computadora en formato pdf.

esta version viene con:

  • una tabla basica de muestra con algunos datos incluidos
  • un formulario para consultas(sin funcionar)

en la proxima version se agregara:

  • una tabla mas completa para cargar los datos
  • funcionalidad en el formulario para las consultas
  • agregar botoncitos de redes sociales en el home
